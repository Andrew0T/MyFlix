This is a movie application, which allows users to search and access information for movie and genre description, directors and image URL's. Users are able to sign up, update their information, create and delete a list of their favourite movies. And delete their user name.



    Essential Features
  • Return a list of ALL movies to the user
  • Return data about a single movie by title to the user, with description, genre, director, image URL, whether it’s featured or not
  • Return data about a genre (description) by name/title (e.g., “Comedy”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister'

API Endpoints

Get list of all Movies /movies GET a JSON object holding data about all the movies
Get single Movie by Title /movies/:title GET a JSON object holding data about all the movies
Get Genre by Name /movies/genre/:Name GET a JSON object holding data about all the movies
Get Director by name /movies/directors/:Name GET a JSON object holding data about all directors
New user registration /users POST a JSON object holding data about all users
Update user info /users/:Username PUT a JSON object holding data about all users
Add movie to favourite list /users/:Username/movies/:movieId POST a JSON object holding data about all users
Remove movie to favourite list /users/:Username/movies/:movieId DELETE :Username, :MovieID a JSON object holding data about all users
User deregistration /users/:Username DELETE :Username A text message advising Users has deleted Username
